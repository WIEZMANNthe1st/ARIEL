# 🎮 אריאל·גיימס — הארקייד האישי

דף אינטרנט אחד (offline, ללא תלות ברשת) עם שישה משחקי מסיבה לבת המצווה של אריאל.

## איך פותחים
פותחים את הקובץ `ariel-games.html` בכל דפדפן (כולל בלי אינטרנט) — נייד, טאבלט או מחשב.
אין צורך בהתקנה, שרת או חיבור לרשת.

## המשחקים

| משחק | איך משחקים |
|---|---|
| 🗣️ **אריאליאס** | מסבירים מילה בלי לומר אותה — כל קבוצה מקבלת טיימר, המילים מוצגות על "קלפים" עם אפשרות למצב תמונות למי שעוד לא קורא/ת |
| ❓ **21 שאלות** | האפליקציה מגרילה דבר סודי, והקבוצה שואלת עד 21 שאלות כן/לא כדי לנחש |
| 🌍 **ארץ עיר** | בכל סיבוב מוגרלים אות וקטגוריה — מי שעונה ראשון/ה מנקד |
| 🧠 **אמת או שקר** | עובדה מוצגת על הקלף — צריך להחליט אם היא נכונה, ואז מגלים את התשובה |
| ⚖️ **זה או זה** | דילמות מצחיקות — בוחרים צד ומסבירים למה, בלי "גם וגם" |
| 🎵 **מצא שיר** | מקבלים מילה, והראשונים לשיר שיר שמכיל אותה זוכים בנקודה |

## מבנה כללי
- **הרכזת (Hub):** מסך ראשי עם כל ששת המשחקים כאריחים לבחירה
- **הגדרת משחק:** מספר קבוצות, שמות קבוצות, ואורך סבב (למשחקים המתאימים)
- **מסך העברה (Handoff):** מעביר את הטלפון לקבוצה הבאה, כולל לוח תוצאות
- **מסך משחק:** טיימר, קלף מתהפך, וכפתורי "הצליחו/דילוג"
- כפתור **🏠 כל המשחקים** תמיד זמין לחזרה לרכזת

## עיצוב
סגנון ארקייד אדום-זהב עם:
- באנר עליון עם קרני אור מסתובבות, נקודות פולקה וכוכבי קומיקס
- כותרת "אריאל!" בזהב עם מסגרת אדומה, בסגנון שלט ניאון
- אריחי משחקים עם מסך צבעוני לכל משחק ופס שם בסגנון סרט כתום
- אנימציות עדינות (ריחוף כרטיס, ניצוצות עולים, זוהר ירוק בלחיצה)

## טכני
- קובץ HTML בודד, גופנים משובצים (Rubik + Varela Round) — עובד גם ללא רשת
- JavaScript וניל, ללא ספריות חיצוניות
- מותאם למובייל (RTL, מגע, מסך מלא)
